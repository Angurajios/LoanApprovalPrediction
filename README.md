# 🏦 Loan Approval Prediction System

A Machine Learning web application that predicts whether a loan applicant is eligible for approval based on financial and personal details.

The application is built using **Python**, **Flask**, **LightGBM**, **HTML/CSS**, and deployed using **Vercel** (Frontend) / Flask Backend.

---
## 🌐 Live Demo

🔗 https://loan-approval-prediction-beta.vercel.app/

---

## 📌 Features

- Predict loan eligibility instantly
- User-friendly web interface
- Machine Learning based prediction
- REST API using Flask
- Responsive UI
- Fast prediction response

## 📊 Input Features

The model uses the following features:

- Age
- Income
- Credit Score
- Years Employed
- Debt Ratio
- Number of Bank Accounts

---

## 🎯 Prediction Output

The application predicts:

- ✅ Eligible
- ❌ Not Eligible
